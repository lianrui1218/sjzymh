# 首页数据质量模块增加维度指标 Spec

## Why
当前首页数据质量模块每个部门卡片只显示三个维度指标（完整性、准确性、及时性），需要增加三个维度指标（唯一性、一致性、有效性），使质量评估更加全面。

## What Changes
- 在每个部门质量卡片的quality-metrics中增加三个新的metric-item
- 新增指标：唯一性、一致性、有效性
- 调整综合达标率计算逻辑（六维度加权）

## Impact
- Affected code: index.html
- Affected CSS: 可能需要调整quality-metrics样式

---

## 修改内容

### 原有指标（3个）
1. 完整性
2. 准确性
3. 及时性

### 新增指标（3个）
4. 唯一性
5. 一致性
6. 有效性

### 综合达标率计算
- 原计算方式：完整性×40% + 准确性×35% + 及时性×25%
- 新计算方式：六维度平均或加权计算
