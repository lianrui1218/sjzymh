# 更新PRD文档数据质量模块 Spec

## Why
根据已完成的HTML页面修改，数据质量详情页面已增加三大质量维度（唯一性、一致性、有效性），现需同步更新产品需求文档，使PRD与实际产品保持一致。

## What Changes
- 更新数据质量模块描述，增加六大质量维度定义
- 增加数据质量详情页面功能说明
- 更新部门质量指标数据表

## Impact
- Affected specs: 易普力数据资产门户定制化产品需求文档.md
- Affected code: 无代码变更，仅文档更新

## ADDED Requirements

### Requirement: 数据质量详情页面
系统应提供数据质量详情页面，展示各部门的数据质量评估情况。

#### Scenario: 用户查看部门质量详情
- **WHEN** 用户访问数据质量详情页面
- **THEN** 系统展示所有部门的质量卡片，每个卡片包含六大质量维度指标

### Requirement: 六大质量维度
系统应支持六大质量维度的评估与展示：

1. **完整性**：数据字段填充完整程度
2. **准确性**：数据内容正确程度
3. **及时性**：数据更新及时程度
4. **唯一性**：数据记录唯一程度
5. **一致性**：数据逻辑一致程度
6. **有效性**：数据格式有效程度

#### Scenario: 质量维度展示
- **WHEN** 用户查看部门质量卡片
- **THEN** 系统展示六个维度的质量指标及综合达标率
