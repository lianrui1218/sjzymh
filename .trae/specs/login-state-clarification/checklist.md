# Checklist

本文档用于验证登录状态与未登录状态数据展示与流程状态梳理的实现是否正确。

## 导航栏状态展示

- [ ] 已登录状态导航栏正确显示用户头像和用户名
- [ ] 已登录状态导航栏正确显示通知图标和未读消息数量角标
- [ ] 已登录状态导航栏用户菜单包含所有菜单项（个人中心、我的申请、我的资源、修改密码、管理后台、退出登录）
- [ ] 未登录状态导航栏正确显示"登录"按钮
- [ ] 未登录状态导航栏隐藏用户头像和用户菜单
- [ ] 未登录状态导航栏隐藏通知图标

## 首页我的待办模块

- [ ] 已登录状态正确展示四个标签页（我的待办、我的申请、我的收藏、已办结）
- [ ] 已登录状态正确展示各标签页的数据内容
- [ ] 未登录状态正确展示登录引导界面
- [ ] 未登录状态不展示标签页和数据
- [ ] 未登录状态登录引导界面包含登录按钮和功能说明

## 数据资源页面

- [ ] 已登录状态资源列表正确展示收藏和申请资源按钮
- [ ] 已登录状态收藏按钮点击后正确切换状态（收藏/已收藏）
- [ ] 已登录状态申请资源按钮点击后正确跳转到申请页面
- [ ] 未登录状态资源列表正确展示"登录后申请"和"登录后收藏"按钮
- [ ] 未登录状态点击操作按钮后正确显示登录提示框

## 资源详情页

- [ ] 已登录状态正确展示基础信息、字段信息、权限信息
- [ ] 已登录状态正确展示使用说明
- [ ] 未登录状态正确展示基础信息
- [ ] 未登录状态正确隐藏字段信息
- [ ] 未登录状态正确隐藏权限信息
- [ ] 未登录状态正确隐藏敏感的使用说明内容

## 数据质量详情页

- [ ] 已登录和未登录状态均可正常访问数据质量详情页
- [ ] 数据质量详情页正确展示各部门质量卡片
- [ ] 数据质量详情页正确展示六大质量维度指标

## 登录提示框

- [ ] 登录提示框正确显示标题和提示文字
- [ ] 登录提示框正确显示登录按钮和取消按钮
- [ ] 登录提示框正确显示功能说明
- [ ] 点击登录按钮正确跳转到泛微OA登录页面
- [ ] 点击取消按钮正确关闭提示框

## 登录流程

- [ ] 点击登录按钮正确跳转到泛微OA登录页面
- [ ] 泛微OA登录成功后正确重定向回数据门户
- [ ] 登录成功后正确获取用户信息
- [ ] 登录成功后正确更新页面状态为已登录
- [ ] 登录成功后正确展示用户相关数据

## 退出登录流程

- [ ] 点击退出登录正确清除本地Token和用户信息
- [ ] 退出登录正确调用泛微OA登出接口
- [ ] 退出登录后正确更新页面状态为未登录
- [ ] 退出登录后正确重定向到首页

## 登录过期处理

- [ ] Token过期后正确检测并清除登录状态
- [ ] Token过期后正确显示登录过期提示框
- [ ] 登录过期提示框正确显示重新登录按钮
- [ ] 重新登录后正确恢复用户操作上下文

## 路由守卫

- [ ] 未登录访问申请页面正确重定向到登录页面
- [ ] 未登录访问管理后台正确重定向到登录页面
- [ ] 非管理员访问管理后台正确显示无权访问提示
- [ ] 非管理员访问管理后台正确重定向到首页

## 后端接口

- [ ] 公开数据接口无需Token验证即可访问
- [ ] 登录数据接口需要Token验证才能访问
- [ ] 敏感数据接口需要Token验证和权限验证才能访问
- [ ] 接口返回统一格式的错误码
- [ ] Token无效时接口返回正确的错误码

## 公开数据访问

- [ ] 未登录状态可正常访问Banner轮播图
- [ ] 未登录状态可正常使用全局搜索功能
- [ ] 未登录状态可正常访问数据资源列表
- [ ] 未登录状态可正常访问数据质量概览卡片
- [ ] 未登录状态可正常访问通知公告列表

## 敏感数据保护

- [ ] 未登录状态无法访问资源字段详细信息
- [ ] 未登录状态无法访问资源权限信息
- [ ] 未登录状态无法访问管理后台
- [ ] 未登录状态无法查看其他用户的个人信息
