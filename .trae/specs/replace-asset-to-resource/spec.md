# 将"资产"修改为"资源" Spec

## Why
根据产品需求，将页面中所有涉及"资产"的文字修改为"资源"，保持产品术语一致性。

## What Changes
- 将所有HTML文件中的"数据资产"修改为"数据资源"
- 将所有"资产"相关文字统一修改为"资源"

## Impact
- Affected files: apply.html, admin.html, assets.html, detail.html, notification-detail.html, notifications.html
- 替换范围：标题、导航、页脚、通知内容中的"资产"文字

---

## 需要修改的文件清单

| 文件名 | 修改内容 |
|-------|---------|
| apply.html | 页脚版权信息 |
| admin.html | 标题、导航栏、通知数据 |
| assets.html | 页面标题、页脚 |
| detail.html | 导航链接、页脚 |
| notification-detail.html | 标题、导航、内容 |
| notifications.html | 标题、导航、通知标题 |

---

## 修改规则

| 原文字 | 修改后 |
|-------|-------|
| 数据资产门户 | 数据资源门户 |
| 数据资产 | 数据资源 |
| 资产 | 资源 |
