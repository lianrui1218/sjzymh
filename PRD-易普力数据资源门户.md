## 二、产品整体定位

### 2.1 产品核心目标

易普力数据资源门户是企业级数据资源统一管理平台，旨在实现：

1. **数据资源统一汇聚**：整合企业各业务系统数据资源，形成统一的数据资源目录
2. **数据权限精准管控**：建立完善的数据申请、审批、授权机制，确保数据安全合规
3. **数据质量持续监控**：构建六大维度数据质量评估体系，推动数据质量持续提升
4. **数据价值高效释放**：促进数据资源跨部门共享，提升数据利用效率

### 2.2 面向用户/角色

| 角色 | 用户群体 | 核心诉求 |
|------|----------|----------|
| 数据使用者 | 业务部门员工 | 查找、申请、使用数据资源 |
| 数据管理者 | 部门数据管理员 | 管理本部门数据资源、处理权限申请 |
| 系统管理员 | 信息技术部人员 | 系统配置、用户管理、公告发布 |
| 数据治理人员 | 数据管理部人员 | 数据质量监控、标准规范管理 |
| 领导层 | 公司管理层 | 数据资产概览、质量状况监控 |

### 2.3 核心价值

| 价值维度 | 具体价值 |
|----------|----------|
| **效率提升** | 数据查找时间缩短70%，申请流程自动化 |
| **安全合规** | 数据权限全程可追溯，满足数据安全合规要求 |
| **质量保障** | 数据质量可视化监控，问题及时发现整改 |
| **资产增值** | 数据资源利用率提升，数据资产价值显性化 |

## 四、详细功能需求

### 4.1 首页模块

#### 4.1.1 Banner轮播

**模块定义**：展示企业数据战略宣传内容，支持自动轮播和手动切换。

**页面入口**：首页顶部

**核心交互流程**：
1. 用户进入首页，自动播放Banner轮播
2. 支持点击底部圆点切换图片
3. 每5秒自动切换下一张

**字段/组件说明**：

| 组件 | 说明 |
|------|------|
| Banner图片 | 支持4张轮播图片 |
| 轮播指示器 | 底部圆点指示当前图片位置 |
| 标题文案 | 主标题：数据驱动价值 创新引领未来 |
| 副标题 | 统一数据资源管理 · 精准权限管控 · 高效协同共享 |

**业务规则**：
- 轮播周期：5秒/张
- 支持手动点击切换
- 鼠标悬停暂停轮播

---

#### 4.1.2 全局搜索

**模块定义**：提供数据资源名称关键词搜索功能，快速定位目标资源。

**页面入口**：首页Banner区域下方

**核心交互流程**：
1. 用户在搜索框输入资源名称关键词
2. 点击"搜索"按钮或按回车键
3. 系统返回匹配的资源列表

**字段/组件说明**：

| 组件 | 类型 | 说明 |
|------|------|------|
| 搜索输入框 | 文本输入 | placeholder提示"搜索资源名称" |
| 搜索按钮 | 按钮 | 点击触发搜索 |

**业务规则**：
- 支持模糊匹配
- 搜索范围：资源名称
- 最少输入1个字符触发搜索

---

#### 4.1.3 我的待办

**模块定义**：展示当前用户待处理的审批事项、申请记录、收藏资源和已办结事项。

**页面入口**：首页左侧模块

**核心交互流程**：
1. 用户进入首页，默认显示"我的待办"标签
2. 点击不同标签切换查看对应内容
3. 点击待办项可跳转详情页处理

**字段/组件说明**：

| 标签 | 说明 | 数据项 |
|------|------|--------|
| 我的待办 | 待处理的审批事项 | 申请标题、提交人、审批节点、时间 |
| 我的申请 | 已提交的申请记录 | 申请标题、状态、审批节点、时间 |
| 我的收藏 | 收藏的资源列表 | 资源名称、类型、所属部门 |
| 已办结 | 已完成的审批事项 | 申请标题、审批结果、审批人、时间 |

**状态标识**：

| 状态 | 样式 | 说明 |
|------|------|------|
| 未读 | 蓝色背景 | 新待办，未查看 |
| 超期 | 红色背景 | 超过处理时限 |
| 审批中 | 蓝色标签 | 申请正在审批流程中 |
| 已通过 | 绿色标签 | 申请已批准 |
| 已驳回 | 红色标签 | 申请被拒绝 |

**业务规则**：
- 待办数量实时显示在标签上
- 超期待办高亮显示
- 默认按时间倒序排列

---

#### 4.1.4 核心数据概览

**模块定义**：展示平台各类数据资源的总量统计。

**页面入口**：首页右侧模块

**字段/组件说明**：

| 指标 | 说明 | 图标颜色 |
|------|------|----------|
| 数据表 | 平台数据表总量 | 蓝色 |
| API | 平台API接口总量 | 绿色 |
| 业务指标 | 平台业务指标总量 | 红色 |

**业务规则**：
- 数据实时更新

---

#### 4.1.5 数据质量概览

**模块定义**：展示各部门数据质量评估情况，包含六大质量维度。

**页面入口**：首页中部模块

**核心交互流程**：
1. 展示各部门质量卡片
2. 显示六大维度评分：完整性、准确性、及时性、唯一性、一致性、有效性
3. 显示综合达标率
4. 点击"查看更多"跳转数据质量详情页

**字段/组件说明**：

| 字段 | 说明 |
|------|------|
| 部门名称 | 数据资源所属部门 |
| 状态标签 | 达标/预警/未达标 |
| 数据资源数 | 该部门数据资源数量 |
| 六大维度 | 完整性、准确性、及时性、唯一性、一致性、有效性 |
| 综合达标率 | 六大维度加权平均分 |

**状态判定规则**：

| 状态 | 条件 | 颜色 |
|------|------|------|
| 达标 | 综合达标率 ≥ 95% | 绿色 |
| 预警 | 90% ≤ 综合达标率 < 95% | 蓝色 |
| 未达标 | 综合达标率 < 90% | 红色 |

```

```
